##NoOkButton

**[NoOkButton]** [>tag:mode]

En una ficha, en el modo indicado no deseamos que aparezca el botón *aceptar*.

####Ejemplos

```

[NoOkButton] a,mR

[Fields]
    Código	| cd_cli	| 0 | T  | 4	|    | *   |	|     | 
    Nombre	| nm_cli	| X | T  | 40	|    | M   |	|  #  | 


```
